# Cypress
This automation is written in TypeScript (a strict syntactical superset of JavaScript and adds optional static typing to the language)

## Running Tests : https://github.com/devynursadiah123/Cypress-typescript/tree/master/cypress/videos

## How to run

```bash
npm install
```

**run cypress**
```bash
npx cypress run
```
![Task 1](https://github.com/devynursadiah123/Cypress-typescript/assets/132911115/1da98879-5f6f-4ef1-8e61-48216070a41c)


**open cypress dashboard**
```bash
npx cypress open
```
